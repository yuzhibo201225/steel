# PLAN

## Objective
Build a minimal, reproducible NEU-DET experiment project for RT-DETR (main) vs YOLO (baseline) under the same 4-fold stratified CV protocol.

## Milestones
1. Data split utility (4-fold stratified, fixed seed).
2. Fold-aware training scripts for RT-DETR and YOLO.
3. Unified metrics schema and evaluation-normalization utility.
4. ANOVA + Tukey export pipeline.
5. Runbook docs + smoke-test commands.
